## FurOS 

**Before continuing, make sure to backup your things on a seperate USB stick. (Highly Recommended!).** 

Thank you for using **FurOS!** A custom operating system made with the operating system Linux!. 

Heres a few things to note:  

***Minimum Storage Recommended: 15 GB of free space (25 GB Recommended).*** 

***RAM Cap: 2GB Minimum (4GB Recommended).***

***Graphics Card: Intel & AMD Intergrated (Highly Recommended Nvidia).***

***Processor: 2.0 GHz dual-core 64-bit processor or better (Intel Core i3/i5/i7/i9, AMD Ryzen 3/5/7/9, or equivalent).***


## Installation Process

Heres a quick guide to install the custom operating system, 

Step One: Install a USB flash Driver Tool (Rufus, Windows Built In Tool, BalenaEtcher). 

Step Two: While the driver tool is installing you will need a blank USB Stick **8GB** Recommeneded. 

Step Three: Once you have sucessfully, mounted the ISO File, onto the USB Stick make sure to follow these instructions below. 

**Start By tapping the BIOS/BOOT menu hotfix keys such as; F12 HP/DELL | F11 MSI | F2/DEL ASUS Acer** 

Lastly: Once you have Sucessfully booted to your USB stick, please follow the on screen instructions. 

 ##  ⚠  WARNING: Before Making this as your __main OS,__ i highly recommened you use the live installer before choosing the install to OS system (Which will remove and replace your current hard disk).
